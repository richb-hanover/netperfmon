netperf.richb-hanover.com

netperf http://netperf.org is a network performance measurement tool. It relies on two programs running on separate computers:

  • netserver that listens for connections from a...
  • netperf client tht connects and tells what to do.
The netperf client can send commands send or receive streams of data and thus measure the data transfer rates between the two devices.

This server hosts a netserver that is publicly accessible. It's open to anyone to use, so it might get heavily loaded.
