netperf.richb-hanover.com

netperf is a network performance measurement tool (http://netperf.org). It relies on two programs running on separate computers:

  • a netserver that listens for commands from ...
  • a netperf client that sends those commands at the user's direction.
The netperf client's commands tell netserver to send or receive streams of data between the two devices and then measures the data transfer rates between them.

Abuse Policy

This server is open to network researchers who need a stable system for testing network protocols and implementations. Individuals are welcome to use the server for occasional network performance tests. We reserve the right to block access if we detect sustained over-use of this service.

Recent netserver activity

This server hosts a netserver that is publicly accessible. It's open to anyone to use, so it may be heavily loaded from time to time. It also has limited bandwidth to certain parts of the Internet (see the About page for details.) This page shows the current and recent load.
