About This Site

This site is a small Node/Express/Handlebars concoction. It examines the creation date of netserver.debug* files from /tmp, then uses the results to populate a Handlebars template. Express provides the backend web routing/handling, and Node.js makes it all go.

Apache2 is running on port 80, and uses a virtual host for netperf.richb-hanover.com to proxy to the node process that's listening on port 3000.

Many thanks to Raymond Camden for his Introduction to Express tutorial which is the basis of this code. Get the source to this project from Github.com.
